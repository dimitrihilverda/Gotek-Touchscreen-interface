Gotek Touchscreen Interface — flasher

Flash the latest firmware to your ESP32-S3 device straight from the browser. No CLI, no Arduino IDE.

Your browser doesn't expose the Web Serial API. Please open this page in Chrome, Edge, or Opera on a desktop OS (Firefox and Safari are not supported).

Install

Pick your board, plug the device into a USB port, then click Install and choose the matching serial port from the list.

latest

The flasher pulls the firmware binaries from the latest GitHub release. First-time installs will erase the flash; existing settings on the SD card are untouched.

Before you start

  • USB-C / USB-A cable that supports data (not power-only).
  • The device must enumerate as a serial port. If nothing shows up, hold BOOT while plugging in, then release.
  • Close the Arduino IDE / any other serial monitor first.

After flashing

  1. Unplug the device, reinsert the SD card with your ADF / DSK collection and a CONFIG.TXT.
  2. Power the device through the Amiga (or any 5V USB source).
  3. On the splash, tap FLIP 180 if the picture is upside-down — the choice is persisted.
  4. Open http://gotek.local/ on the same LAN after WiFi is configured.
